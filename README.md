# English Level Quiz (300 Questions)

A premium English proficiency quiz web application featuring 300 interactive questions, a modern glassmorphism UI, and audio feedback.

## Features
- **300 Questions**: Comprehensive coverage of Tenses, Subjunctive, Should Deletion, Verbals, and more.
- **Modern UI**: Dark mode with glassmorphism effects and smooth animations.
- **Audio Feedback**: Text-to-speech ("Correct!") and sound effects for right answers.
- **Detailed Explanations**: Immediate feedback with Korean explanations for every question.

## Tech Stack
- **Framework**: Vite + React
- **Styling**: Vanilla CSS (Custom Glassmorphism)
- **Deployment**: Optimized for Vercel

## How to Deploy
1. Push this code to GitHub.
2. Link the repository to Vercel.
3. Vercel will automatically detect Vite and deploy.
4. Share the generated `.vercel.app` URL with others.
